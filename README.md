# Muhammad Tanveer - Portfolio App

A modern, responsive personal portfolio web application built with **Flutter** to showcase skills, projects, and professional experience. This application serves as a digital resume and professional presence for internship and job opportunities.

## 📱 Project Overview

This portfolio web application is designed to provide a complete professional presence online. It features a modern UI with dark/light theme support, responsive design for all devices, and interactive components to engage visitors. The application highlights Muhammad Tanveer's skills in Flutter, Firebase, MySQL, Java, C++, and Full Stack Development.

## ✨ Features

- ✅ **Responsive Design** - Optimized for mobile, tablet, and desktop
- ✅ **Dark/Light Theme** - User preference-based theming
- ✅ **Interactive Navigation** - Smooth page transitions
- ✅ **About Section** - Professional summary and personal info
- ✅ **Skills Showcase** - Categorized technical skills
- ✅ **Project Portfolio** - Featured projects with GitHub links
- ✅ **Contact Form** - Direct messaging functionality
- ✅ **Social Media Integration** - LinkedIn, GitHub, Email, Phone
- ✅ **Animated UI** - Smooth fade-in animations
- ✅ **Material Design** - Clean, modern interface
